Homework assignments

Students will use R Markdown to complete their homework assignments, which will allow you to easily save and execute R code, and combine the output with additional text and equations into a single pdf file. Each home assignment will have it’s own R Markdown template for students to fill out and submit. We will use RStudio as an IDE for working with R and authoring R Markdown documents. Those students who are unfamiliar with R Markdown should read through this introduction.
